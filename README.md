# Сайт Екатерины Чибиревой

Готовый статический многостраничный фронтенд для сайта-воронки коротких онлайн-тренировок.

## Как открыть

Откройте `index.html` в браузере. Все страницы используют общие файлы `styles.css` и `app.js`.

## Страницы

- `index.html` — главная
- `how.html` — как это работает
- `formats.html` — форматы работы
- `audience.html` — для кого
- `method.html` — метод
- `results.html` — отзывы и результаты
- `trainer.html` — о тренере
- `contacts.html` — контакты

## Что заменить перед публикацией

- Фото лежат в папке `assets` и названы `trainer-06.jpg` ... `trainer-09.jpg`.
- Контакты и плейсхолдеры мессенджеров находятся в `app.js`, функция `contactsPage`.
- Формы заявки на сайте нет: все CTA-кнопки ведут во ВК `https://vk.com/katrin_kr`.
